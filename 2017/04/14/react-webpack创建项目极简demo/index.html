0、准备工作

既然要用轮子当然就要先安装了，不多逼逼。

1 安装wenpack

$ npm install webpack -g 　　　// 当然前提你要安装node

2 安装webpack-server（webpack的服务器，不用请忽略）

$ npm install webpack-dev-g

3 安装react react-dom到项目

$ npm install react react-dom –save

4 安装 Babel 的 loader 以支持 ES6 语法:

$ npm install babel-core babel-loader babel-preset-es2015 babel-preset-react –save-dev
